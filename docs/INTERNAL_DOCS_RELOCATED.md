# Internal documentation boundary

Public architecture and decisions that govern AKDB itself live in
[docs/architecture](architecture/) and [docs/adr](adr/).

Private implementation planning, cross-project architecture, contracts, and the maintainer
runbook live in the private Tiny Tool Development repository, including:

- `Git/docs/superpowers/specs/architectural-knowledge-db/`
- `Git/docs/superpowers/plans/architectural-knowledge-db/`
- `Git/docs/ADR/` for cross-project decisions

This public repository does not contain imported project corpora, generated exports, copied
SAD/UML from other repositories, credentials, or runtime databases.
